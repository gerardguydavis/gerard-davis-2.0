About Me

I’m a front-end developer living in Syracuse, NY. I graduated with a Bachelor of Arts in Media, Society, and The Arts from Purchase College and started coding in February 2021. I work with a mobile-first approach to create clean, responsive websites that look great on any device. My work is entirely hand-coded, and I aim to write code that is thoughtful and easy to understand.

When I’m not coding, you can usually find me in the kitchen baking biscotti or, more likely, on the couch playing Dragon Quest with my cats, Tetris and Boombox.

Donut Drop

Donut Drop is a classic match 3 game built primarily with vanilla JavaScript. It's a time-limited game featuring functions to click, drag and swap donuts, prohibit invalid moves (Swaps can only be made with up, down, left, and right adjacent donuts and must create a match of 3 or more), log and announce high scores, start new games with a randomized board, and display possible valid moves using a hint button.

Languages: HTML, SCSS, JavaScript
